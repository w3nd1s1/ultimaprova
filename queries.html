🔎 SPARQL Queries – Villa Lante

This section collects a set of SPARQL queries used to explore data related to Villa Lante through the ARCO knowledge graph.

1. Search for resources labeled “Villa Lante”
Query to identify all resources whose label contains "Villa Lante". 
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?bene ?label
WHERE {
  ?bene rdfs:label ?label .
  FILTER(REGEX(?label, "Villa Lante", "i"))
}
LIMIT 50
2. Creators and agents of the cultural property
Extracts information about authors, architects, and involved institutions. 
SELECT DISTINCT ?property ?value ?optionalInfo
WHERE {
  <https://w3id.org/arco/resource/CulturalPropertyPart/1200088359-part-villa-lante-a-bagnaia-monumento-nazionale-v-an>
    ?property ?value .

  OPTIONAL {
    ?value ?optionalInfo ?x .
  }

  FILTER(
    REGEX(STR(?property),
    "agent|author|creator|architect|institution",
    "i")
  )
}
3. Restoration and conservation interventions
Filters information related to maintenance, restoration, and preservation activities. 
SELECT DISTINCT ?property ?value
WHERE {
  <https://w3id.org/arco/resource/CulturalPropertyPart/1200088359-part-villa-lante-a-bagnaia-monumento-nazionale-v-an>
    ?property ?value .

  FILTER(
    REGEX(STR(?property),
    "restor|conserv|maintenance|repair|preserv",
    "i")
  )
}
4. Events and related situations
Retrieves events or situations associated with the cultural property. 
SELECT DISTINCT ?property ?value
WHERE {
  <https://w3id.org/arco/resource/CulturalPropertyPart/1200088359-part-villa-lante-a-bagnaia-monumento-nazionale-v-an>
    ?property ?value .

  FILTER(
    REGEX(
      STR(?property),
      "event|situation",
      "i"
    )
  )
}
5. Combined view (Agents + Events + Conservation)
Unified SPARQL query using UNION to retrieve agents, events, and conservation-related information in a single structured result set. 
SELECT DISTINCT ?property ?value
WHERE {

  {
    <https://w3id.org/arco/resource/CulturalPropertyPart/1200088359-part-villa-lante-a-bagnaia-monumento-nazionale-v-an>
      ?property ?value .

    FILTER(
      REGEX(STR(?property),
      "agent|author|creator|architect|institution",
      "i")
    )
  }

  UNION

  {
    <https://w3id.org/arco/resource/CulturalPropertyPart/1200088359-part-villa-lante-a-bagnaia-monumento-nazionale-v-an>
      ?property ?value .

    FILTER(
      REGEX(STR(?property),
      "event|situation",
      "i")
    )
  }

  UNION

  {
    <https://w3id.org/arco/resource/CulturalPropertyPart/1200088359-part-villa-lante-a-bagnaia-monumento-nazionale-v-an>
      ?property ?value .

    FILTER(
      REGEX(STR(?property),
      "restor|conserv|maintenance|repair|preserv",
      "i")
    )
  }

}
